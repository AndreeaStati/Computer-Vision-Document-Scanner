# Document Scanning, Rectification and Quality Assessment

## 🔧 Instalare și rulare

1.  Instalează Python 3.10+

2.  Clonează repo-ul:

```bash
git clone https://github.com/SVA-2026/sva-project-scanningapp.git
cd sva-project-scanningapp/
```

3.  Creează și activează un mediu virtual

```bash
python3 -m venv venv
source venv/bin/activate
```

4.  Instalează pachetele necesare pentru OpenCV si plotare

```bash
pip install opencv-python
pip install matplotlib
```

5.  Ruleaza aplicatia

```bash
python main.py
```
